# Nvim Cheatsheet

## Pluginy
| Komenda | Opis |
|---|---|
| `:PlugUpdate` | Aktualizuj wszystkie pluginy |
| `:PlugClean` | Usuń nieużywane pluginy |
| `:PlugList` | Lista zainstalowanych pluginów |

## Ogólne
| Skrót | Opis |
|---|---|
| `Space+?` | Pokaż wszystkie skróty |
| `Space+a` | Zaznacz wszystko |
| `Space+m` | Menu (dashboard) |
| `Space+q` | Zamknij buffer |

## Pliki i wyszukiwanie
| Skrót | Opis |
|---|---|
| `Space+e` | Eksplorator plików (oil) |
| `Space+ff` | Znajdź plik |
| `Space+fw` | Znajdź słowo |
| `Space+fh` | Znajdź pomoc |
| `Space+fc` | Znajdź w configu |

## Taby
| Skrót | Opis |
|---|---|
| `Space+t` | Nowy tab |
| `Space+x` | Zamknij tab |
| `Space+j` | Następny tab |
| `Space+k` | Poprzedni tab |

## Buffery
| Skrót | Opis |
|---|---|
| `Tab` | Następny buffer |
| `Shift+Tab` | Poprzedni buffer |

## Splity
| Skrót | Opis |
|---|---|
| `Space+v` | Podział pionowy |
| `Space+s` | Podział poziomy |
| `Ctrl+h` | Przejdź do lewego splita |
| `Ctrl+l` | Przejdź do prawego splita |
| `Ctrl+j` | Przejdź do dolnego splita |
| `Ctrl+k` | Przejdź do górnego splita |
| `Ctrl+←` | Zwiększ szerokość splita |
| `Ctrl+→` | Zmniejsz szerokość splita |

## Git
| Skrót | Opis |
|---|---|
| `Space+gg` | Git status |
| `Space+gc` | Git branches |

## Markdown
| Skrót | Opis |
|---|---|
| `Space+pp` | Podgląd markdown |
| `Space+pf` | Formatuj z Prettier |

## Wcięcia (tryb visual)
| Skrót | Opis |
|---|---|
| `>` | Wcięcie w prawo |
| `<` | Wcięcie w lewo |
